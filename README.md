# Docker-example
Окружение для разработки сайтов

# Команды:

Запуск контейнеров в фоне.

`docker-compose up -d`

Остановка и удаление контейнеров.

`docker-compose down`

Остановка контейнеров.

`docker-compose stop`

Удаление контейнеров.

`docker-compose rm`

Удаление образов.

`docker image prune -a`
